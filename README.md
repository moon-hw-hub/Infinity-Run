장애물을 점프와 낙하를 이용해 피해가면서 과일을 최대한 많이 먹는 것을 목표로 하는 게임입니다.

- 난이도 상승<br>
젤리를 150개 먹을 때마다 게임의 진행 속도가 빨라지고 스폰되는 젤리의 종류가 달리집니다!

|체리|사과|멜론|오렌지|파인애플|
|:---:|:---:|:---:|:---:|:---:|
|1점|3점|5점|7점|9점|

- 게임 오버<br>
주인공이 장애물에 부딪치면 게임 오버입니다.
